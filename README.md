# Twitter_Scroll_Meter  
Twitter/Xのスクロール量を計測して表示するChrome拡張  

Chromeウェブストアから最新版がDLできます。  
https://chromewebstore.google.com/detail/twitter-scroll-meter/aifgbefapjkjojljkcodnacdmenpdcfc?authuser=4&hl=ja&pli=1  


[主な機能]  
・PPI計算  
・常時計測/リアルタイム表示(m)  
・計測タイミング設定  
・計測頻度設定  
・補正係数設定  


# 使い方  
入れたらPPIを設定してね
